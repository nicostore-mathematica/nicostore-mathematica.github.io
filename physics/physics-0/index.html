Part 1 绪论

· 单位制

为了进行测量，人们必须引进一些基本的测量单位。在物理学中，有三个基本的物理量：

  • 长度 (Length)，其单位为厘米 (Centimeter);
  • 时间 (Time)，其单位为秒 (Second);
  • 质量 (Mass)，其单位为克 (Gram)。

它们被称为基本单位。物理学中用到的其它量都可以用这三个基本物理量的适当组合给出。精确一点讲，任何一个物理量 QQQ 的单位都可以写作

[Q]=[L]α[M]β[T]γ\begin{equation} [Q] = [L]^{\alpha}[M]^{\beta}[T]^{\gamma} \end{equation} [Q]=[L]α[M]β[T]γ​​

的形式。这里，常数 α\alphaα, β\betaβ 和 γ\gammaγ 被称为 QQQ 的量纲指数。

/example/ 我们知道，一个匀速运动的物体速率 vvv 可以定义为

v=LT.\begin{equation} v = \frac{\text{L}}{\text{T}}. \end{equation} v=TL​.​​

因此，我们有

[v]=[L][T]=[L]1[M]0[T]−1.\begin{equation} [v] = \frac{[L]}{[T]} = [L]^{1}[M]^{0}[T]^{-1}. \end{equation} [v]=[T][L]​=[L]1[M]0[T]−1.​​

既 α=1\alpha = 1α=1, β=0\beta = 0β=0, γ=−1\gamma = -1γ=−1 。

/example/ 一个匀加速运动的物体的加速度可以定义为

a=ΔvΔT.\begin{equation} a = \frac{\Delta \text{v}}{\Delta \text{T}}. \end{equation} a=ΔTΔv​.​​

因此，我们有

[a]=[v][T]=[L]1[M]0[T]−2.\begin{equation} [a] = \frac{[v]}{[T]} = [L]^{1}[M]^{0}[T]^{-2}. \end{equation} [a]=[T][v]​=[L]1[M]0[T]−2.​​

即 α=1\alpha = 1α=1, β=0\beta = 0β=0, γ=−2\gamma = -2γ=−2 。

/example/ 电荷的量纲，根据 Coulomb 定律，我们有

F=e2r2.\begin{equation} F = \frac{e^{2}}{r^{2}}. \end{equation} F=r2e2​.​​

这里，FFF 为两个相距为 rrr 的带有相同电荷 eee 的带电体之间的 Coulomb 斥力。因此，我们有

[F]=[e]2[L]−2.\begin{equation} [F] = [e]^2[L]^{-2}. \end{equation} [F]=[e]2[L]−2.​​

另一方面，从牛顿第二定律

F=ma,\begin{equation} F = ma, \end{equation} F=ma,​​

我们又有

[F]=[M][a]=[M][L][T]−2.\begin{equation} [F] = [M][a] = [M][L][T]^{-2}. \end{equation} [F]=[M][a]=[M][L][T]−2.​​

这里，我们使用了上一个例子中推得的加速度的量纲表达式。因此，我们得到

[M][L][T]−2=[e]2[L]−2.\begin{equation} [M][L][T]^{-2} = [e]^2[L]^{-2}. \end{equation} [M][L][T]−2=[e]2[L]−2.​​

由此，我们解得

[e]2=[M][L]3[T]−2,\begin{equation} [e]^2 = [M][L]^3[T]^{-2}, \end{equation} [e]2=[M][L]3[T]−2,​​

或是

[e]=[L]3/2[M]1/2[T]−1.\begin{equation} [e] = [L]^{3/2}[M]^{1/2}[T]^{-1}. \end{equation} [e]=[L]3/2[M]1/2[T]−1.​​

因此，我们有 α=3/2\alpha = 3/2α=3/2, β=1/2\beta = 1/2β=1/2, γ=−1\gamma = -1γ=−1 。

· 物质及其相互作用

物理学是研究物质的形态及其相互作用的科学。

就物质的形态而言，目前已知其最小的单元是所谓夸克 (Quark)，轻子 (Lepton) 和中微子 (Neutrino) 。其中，夸克有 12 种。它们分别是：

(i) 上夸克 (up-quark)，带电荷量为 q=23q = \frac{2}{3}q=32​ 及反上夸克 (anti-up-quark)，带电荷量为 q=−23q = -\frac{2}{3}q=−32​;

(ii) 下夸克 (down-quark)，带电荷量为 q=−13q = -\frac{1}{3}q=−31​ 及反下夸克 (anti-down-quark)，带电荷量为 q=13q = \frac{1}{3}q=31​;

(iii) 奇异夸克 (strange-quark)，带电荷量为 q=−13q = -\frac{1}{3}q=−31​ 及反奇异夸克 (anti-strange-quark)，带电荷量为 q=13q = \frac{1}{3}q=31​;

(iv) 粲夸克 (charm-quark)，带电荷量为 q=23q = \frac{2}{3}q=32​ 及反粲夸克 (anti-charm-quark)，带电荷量为 q=−23q = -\frac{2}{3}q=−32​;

(v) 顶夸克 (top-quark)，带电荷量为 q=−13q = -\frac{1}{3}q=−31​ 及反顶夸克 (anti-top-quark)，带电荷量为 q=13q = \frac{1}{3}q=31​;

(vi) 底夸克 (bottom-quark)，带电荷量为 q=23q = \frac{2}{3}q=32​ 及反底夸克 (anti-bottom-quark)，带电荷量为 q=−23q = -\frac{2}{3}q=−32​。

相应地，我们有如下的轻子和中微子。

(i) 电子 (electron)，带电荷量为 q=−1q = -1q=−1 及反电子 (anti-electron)，带电荷量为 q=1q = 1q=1。相对应的中微子为电子中微子 νe\nu_eνe​ 和反电子中微子 νˉe\bar{\nu}_eνˉe​，它们的带电荷量皆为零；

(ii) μ\muμ 介子 (muon)，带电荷量为 q=−1q = -1q=−1 及反 μ\muμ 介子 (anti-muon)，带电荷量为 q=1q = 1q=1。相对应的中微子为 μ\muμ 介子中微子 νμ\nu_\muνμ​ 和反 μ\muμ 介子中微子 νˉμ\bar{\nu}_\muνˉμ​，它们的带电荷量皆为零；

(iii) τ\tauτ 介子，带电荷量为 q=−1q = -1q=−1 及反 τ\tauτ 介子，带电荷量为 q=1q = 1q=1。相对应的中微子为 τ\tauτ 介子中微子 ντ\nu_\tauντ​ 和反 τ\tauτ 介子中微子 νˉτ\bar{\nu}_\tauνˉτ​，它们的带电荷量皆为零。

这些粒子是我们这个世界的基本建筑材料。我们过去所熟知的基本粒子，例如质子 (Proton)，中子 (Neutron) 和 Λ\LambdaΛ 粒子等等，都是由夸克组成的。例如，我们有

P=uud,N=udd,Λ0=uds.P = uud, \quad N = udd, \quad \Lambda^0 = uds. P=uud,N=udd,Λ0=uds.

有趣的是，到目前为止，人们尚不能在实验上观察到游离状态的夸克。这就促使人们发展了所谓的夸克禁闭理论。

在此基础上，质子、中子和电子的不同组合，又给出了大约 110 种不同的原子（不包含同位素）。再由原子构成不可计数的分子。而我们人类的细胞则是由许多巨分子构成的。总之，就我们今天所知而言，这个世界就是由上述的几种夸克、轻子以及中微子构造而成的。

当然，要使得这些基本单元能够束缚在一起，构成中子和质子，它们之间必须存在相互作用。目前人们普遍认为，夸克之间是由所谓胶子 (Gluon) 来传递相互作用的。目前这一理论仍在发展之中，我们就不再多述了。

再往上面一个层次，让我们来看一看质子、中子、介子等等这些被称为强子的粒子之间的相互作用以及强子与轻子、轻子与中微子之间的相互作用。

到目前为止，人们只找到了粒子之间的四种基本相互作用，而物质之间的其它相互作用形式都不过是这四种基本相互作用在不同条件下的表现而已。它们是

(1) 强相互作用。这是一种仅存在于强子之间的相互作用，作用长度大约为 10−1310^{-13}10−13 厘米。为了与其它相互作用相比较，我们将它的强度取作 1。

(2) 弱相互作用。这种相互作用存在于轻子与其中微子之间，以及轻子与某些强子之间。这也是一种短程相互作用，其作用长度大约为 10−1610^{-16}10−16 厘米，而其作用强度为 10−1310^{-13}10−13。

(3) 电磁相互作用。这种相互作用，也称库仑相互作用，存在于带电粒子之间。它可以被写成如下的形式

F=q1q2r3r.\begin{equation} \mathbf{F} = \frac{q_1 q_2}{r^3} \mathbf{r}. \end{equation} F=r3q1​q2​​r.​​

这是一种长程力，其作用强度为 10−210^{-2}10−2。

(4) 引力相互作用。存在于具有质量的物体之间。它具有如下的形式

F=−Gm1m2r3r.\begin{equation} \mathbf{F} = -\frac{G m_1 m_2}{r^3} \mathbf{r}. \end{equation} F=−r3Gm1​m2​​r.​​

这是一种长程力，其作用强度为 10−3810^{-38}10−38。

/example/ 当宇宙飞船脱离地球的大气层后，将暴露在宇宙射线的辐射之中。这些射线是由大量能量很高的粒子组成的。其中有些带电，有些不带电。从对于人体的伤害的角度考虑问题，我们应该主要防范哪些粒子呢？

从上面''的介绍中，我们知道，不带电粒子与构成我们肌体的物质之间的相互作用只能是引力，强相互作用或是弱相互作用。而带电粒子则可通过电磁相互作用破坏肌体中的大分子链。显然，与电磁相互作用相比，我们可以略去引力和弱相互作用的影响不计。至于强相互作用，尽管其强度较大，但其作用长度很短。因此，在宇宙线入射密度不是很大的情况下，破坏作用并不大。因此，我们主要应该防范带电粒子对于人体的伤害。

· 物理世界的尺度层次

在物理学的研究中，主要是采用简约主义 (Reductionism) 的做法。也就是说，在处理一个复杂问题时，我们要区别哪些因素是起主要作用的，而哪些因素可以略去不计，以求达到我们对于物质运动规律的本质上的理解。

例如，当我们讨论地球的运动时，我们显然没有必要考虑地面上一个人是快跑还是慢行所带来的影响。相对于地球而言，人的尺度太小了。由此我们看到，在探讨哪些因素比较重要时，首先比较一下这些因素的长度、质量及时间尺度的量级是必须做的一件事情。

事实上，有些时候，有了有关的量级的概念，再加上一些基本常识，我们就可以对于一些物理现象的本质有一个非常直观的认识。

/example/ 试问一个爆竹爆炸时所释放出来的能量大约是多少？

/Claim/

我们知道，火药的燃烧是一种化学反应。这个反应过程是在分子层次上进行的。

众所周知，原子的半径是 10−810^{-8}10−8 厘米的量级。这也就是物质处于固态或液态时内部分子之间的大致的平均距离。

在这种距离尺度内，粒子之间强相互作用和弱相互作用的影响可以略去不计。而引力相互作用的影响又太弱了。因此，电磁相互作用是引起化学反应的主要因素。

与之相关的能量的数量级为

ϵ=e2r0≈e210−8cm.\begin{equation} \epsilon = \frac{e^2}{r_0} \approx \frac{e^2}{10^{-8} \text{cm}}. \end{equation} ϵ=r0​e2​≈10−8cme2​.​​

作为一个基本物理常数，应该记住，在克 - 厘米 - 秒制中，电子电荷为

e≈4.8×10−10.\begin{equation} e \approx 4.8 \times 10^{-10}. \end{equation} e≈4.8×10−10.​​

因此，我们有

ϵ≈10−1910−8erg=10−11erg.\begin{equation} \epsilon \approx \frac{10^{-19}}{10^{-8}}\text{erg} = 10^{-11} \text{erg}. \end{equation} ϵ≈10−810−19​erg=10−11erg.​​

这也是两个火药分子发生化学反应时释放出来的能量。在一克分子火药中，大约有 102310^{23}1023 个分子。因此，爆炸所释放的能量大约为

E=ϵ×1023=10−11×1023=1012erg=105J=1054.184Cal≈104Cal.\begin{equation} E = \epsilon \times 10^{23} = 10^{-11} \times 10^{23} = 10^{12} \text{erg} = 10^5 \text{J} = \frac{10^5}{4.184} \text{Cal} \approx 10^4 \text{Cal}. \end{equation} E=ϵ×1023=10−11×1023=1012erg=105J=4.184105​Cal≈104Cal.​​

Part 2 数学准备

· 向量代数

在物理学的计算中，我们遇到的最多的是标量和向量。所谓标量，就是具有正或负号的实数，以及为了运算方便起见而引入的复数。而向量则是具有方向的物理量，通常用 A⃗\vec{A}A 或 A\mathbf{A}A 来表示。它的长度记作 ∣A∣|\mathbf{A}|∣A∣，而

eA=A∣A∣\begin{equation} \mathbf{e}_\mathbf{A} = \frac{\mathbf{A}}{|\mathbf{A}|} \end{equation} eA​=∣A∣A​​​

则表示沿着 A\mathbf{A}A 的方向的单位向量。

在中学物理的学习中，我们曾经学过如何用平行四边形法则来计算两个向量 A\mathbf{A}A 和 B\mathbf{B}B 之和。首先，我们将两个向量的起点通过平移，移动到同一点。然后以这两个向量为边，做一个平行四边形。这个平行四边形的对角线就是我们所要求的矢量和 C\mathbf{C}C。记作

A+B=C.\begin{equation} \mathbf{A} + \mathbf{B} = \mathbf{C}. \end{equation} A+B=C.​​

对于向量和，不难验证，我们有

A+B=B+A,\begin{equation} \mathbf{A} + \mathbf{B} = \mathbf{B} + \mathbf{A}, \end{equation} A+B=B+A,​​

以及

(A+B)+C=A+(B+C).\begin{equation} (\mathbf{A} + \mathbf{B}) + \mathbf{C} = \mathbf{A} + (\mathbf{B} + \mathbf{C}). \end{equation} (A+B)+C=A+(B+C).​​

其次，我们可以定义一个标量 α\alphaα 和一个向量 A\mathbf{A}A 的乘积为

α⋅A=α∣A∣⋅A∣A∣=(α∣A∣)eA.\begin{equation} \alpha \cdot \mathbf{A} = \alpha |\mathbf{A}| \cdot \frac{\mathbf{A}}{|\mathbf{A}|} = (\alpha |\mathbf{A}|) \mathbf{e}_\mathbf{A}. \end{equation} α⋅A=α∣A∣⋅∣A∣A​=(α∣A∣)eA​.​​

特别是当 α=−1\alpha = -1α=−1 时，我们有

(−1)⋅A=−A.\begin{equation} (-1) \cdot \mathbf{A} = -\mathbf{A}. \end{equation} (−1)⋅A=−A.​​

这是一个长度与 A\mathbf{A}A 相同，但方向相反的向量。

显然，用几何做图的方法来求两个向量的和是比较费力的。为了简化计算，我们可以引入各种坐标系。最常用的是所谓直角坐标系。为此，我们取一个原点及从原点引出的三个互相垂直的长度为一的向量，分别记作 i,j\mathbf{i}, \mathbf{j}i,j 和 k\mathbf{k}k。再将每个单位向量延长为一个实数轴，分别称为 x,yx, yx,y 和 zzz 轴。现在，对于空间中的任何一点 P\mathbf{P}P，我们可以将它到原点的距离向量 r\mathbf{r}r 写作

r=xi+yj+zk.\begin{equation} \mathbf{r} = x \mathbf{i} + y \mathbf{j} + z \mathbf{k}. \end{equation} r=xi+yj+zk.​​

它被称为空间向量的直角坐标表达方式，或是 r\mathbf{r}r 的直角坐标分解式。

建立了直角坐标系后，我们就可以将向量的加法和数乘转化为代数运算了。首先，我们有

α⋅r=α⋅(xi+yj+zk)=(αx)i+(αy)j+(αz)k,\begin{equation} \alpha \cdot \mathbf{r} = \alpha \cdot (x \mathbf{i} + y \mathbf{j} + z \mathbf{k}) = (\alpha x) \mathbf{i} + (\alpha y) \mathbf{j} + (\alpha z) \mathbf{k}, \end{equation} α⋅r=α⋅(xi+yj+zk)=(αx)i+(αy)j+(αz)k,​​

以及

r1+r2=(x1i+y1j+z1k)+(x2i+y2j+z2k)=(x1+x2)i+(y1+y2)j+(z1+z2)k.\begin{equation} \mathbf{r}_1 + \mathbf{r}_2 = (x_1 \mathbf{i} + y_1 \mathbf{j} + z_1 \mathbf{k}) + (x_2 \mathbf{i} + y_2 \mathbf{j} + z_2 \mathbf{k}) = (x_1 + x_2) \mathbf{i} + (y_1 + y_2) \mathbf{j} + (z_1 + z_2) \mathbf{k}. \end{equation} r1​+r2​=(x1​i+y1​j+z1​k)+(x2​i+y2​j+z2​k)=(x1​+x2​)i+(y1​+y2​)j+(z1​+z2​)k.​​

除了直角坐标系之外，还可以根据具体问题的要求引进其它的坐标系。

在这门课程中，我们经常用到的另外一种坐标系是所谓极坐标系。

首先，我们选取空间中的一点为坐标原点 OOO，又任取一从原点出发的轴作为极轴。

在这一坐标系中，我们令空间中一点 PPP 到原点的连线方向为矢径方向 er\mathbf{e}_rer​，连线 OP‾\overline{OP}OP 的长度为第一个坐标 rrr。

同时，令连线 OP‾\overline{OP}OP 与极轴的夹角 $ \theta $ 为第二个坐标。相应地，我们将垂直于 er\mathbf{e}_rer​，指向夹角 $ \theta $ 增大方向的单位向量取为 eθ\mathbf{e}_\thetaeθ​，称为坐标 $ \theta $ 的方向。

最后，利用右手螺旋法则，定出一个同时垂直于 er\mathbf{e}_rer​ 和 eθ\mathbf{e}_\thetaeθ​ 的单位向量 ek\mathbf{e}_kek​。这样得到的坐标系即所谓的极坐标系。它非常适合用来描述在中心力场中物体的运动。

其次，我们还可以引入两个向量的内积的定义

r1⋅r2=(x1i+y1j+z1k)⋅(x2i+y2j+z2k)=x1x2+y1y2+z1z2.\begin{equation} \mathbf{r}_1 \cdot \mathbf{r}_2 = (x_1 \mathbf{i} + y_1 \mathbf{j} + z_1 \mathbf{k}) \cdot (x_2 \mathbf{i} + y_2 \mathbf{j} + z_2 \mathbf{k}) = x_1 x_2 + y_1 y_2 + z_1 z_2. \end{equation} r1​⋅r2​=(x1​i+y1​j+z1​k)⋅(x2​i+y2​j+z2​k)=x1​x2​+y1​y2​+z1​z2​.​​

特别是当 r1⋅r2=0\mathbf{r}_1 \cdot \mathbf{r}_2 = 0r1​⋅r2​=0 成立时，我们称两个向量是垂直的。

我们还可以引入向量的直积

两个矢量 A\mathbf{A}A 和 B\mathbf{B}B 的直积定义作

A×B=C,∣C∣=∣A∣⋅∣B∣⋅sin⁡ϕ.\begin{equation} \mathbf{A} \times \mathbf{B} = \mathbf{C}, \quad |\mathbf{C}| = |\mathbf{A}| \cdot |\mathbf{B}| \cdot \sin \phi. \end{equation} A×B=C,∣C∣=∣A∣⋅∣B∣⋅sinϕ.​​

这里，ϕ\phiϕ 为矢量 A\mathbf{A}A 和 B\mathbf{B}B 之间的夹角。而 C\mathbf{C}C 的方向则由右手螺旋法则来决定。

根据这一定义，我们不难验证下面的关系成立。

(αA)×B=α(A×B),A×B=−B×A,\begin{equation} (\alpha \mathbf{A}) \times \mathbf{B} = \alpha (\mathbf{A} \times \mathbf{B}), \quad \mathbf{A} \times \mathbf{B} = -\mathbf{B} \times \mathbf{A}, \end{equation} (αA)×B=α(A×B),A×B=−B×A,​​

以及

(A1+A2)×B=A1×B+A2×B.\begin{equation} (\mathbf{A}_1 + \mathbf{A}_2) \times \mathbf{B} = \mathbf{A}_1 \times \mathbf{B} + \mathbf{A}_2 \times \mathbf{B}. \end{equation} (A1​+A2​)×B=A1​×B+A2​×B.​​
